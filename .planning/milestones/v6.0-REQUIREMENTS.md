## v6.0 Requirements

Reading & Dictionary. Gives students a Chinese text reader with instant dictionary lookup, dual Cantonese/Mandarin pronunciation, and lesson integration.

### Dictionary Infrastructure

- [x] **DICT-01**: Dictionary entries table seeded with CC-CEDICT (~124K entries) containing traditional, simplified, pinyin, and English definitions
- [x] **DICT-02**: Dictionary entries enriched with CC-Canto jyutping data for Cantonese pronunciation
- [x] **DICT-03**: Character data table seeded with Make Me a Hanzi (~9K characters) containing radical, decomposition, etymology, and stroke paths
- [x] **DICT-04**: Dictionary lookup API returns entries by character/word match (traditional or simplified) with pinyin and jyutping
- [x] **DICT-05**: Character detail API returns radical breakdown, etymology, decomposition, and stroke data for individual characters
- [x] **DICT-06**: Dictionary entries flagged as Cantonese-only, Mandarin-only, or shared based on CC-CEDICT vs CC-Canto source data

### Chinese Reader

- [x] **READ-01**: Chinese Reader page at /dashboard/reader with text paste area and file import (.txt/.pdf)
- [x] **READ-02**: Imported text segmented into hoverable/tappable word spans using Intl.Segmenter
- [x] **READ-03**: Annotation mode selector toggles between Cantonese (Jyutping), Mandarin (Pinyin), and Plain display
- [x] **READ-04**: Traditional <-> Simplified conversion toggle using opencc-js with HK variant support
- [x] **READ-05**: File import handles multiple Chinese text encodings (UTF-8, GB2312, Big5) via jschardet detection
- [x] **READ-06**: Sentence-level English translation (AI-generated) with tap-to-reveal pattern per sentence
- [x] **READ-07**: Mandarin tone sandhi display shows actual spoken tones using pinyin-pro

### Character Popup

- [x] **POPUP-01**: Single shared popup component positioned via Floating UI, triggered by character hover (desktop) or tap (mobile)
- [x] **POPUP-02**: Popup displays English definitions, pinyin with tone marks, and jyutping pronunciation
- [x] **POPUP-03**: Popup shows radical/component breakdown with etymology information
- [x] **POPUP-04**: Popup includes example words containing the selected character
- [x] **POPUP-05**: Tone comparison layout shows Mandarin pinyin and Cantonese jyutping side-by-side highlighting tonal/phonetic similarities and differences
- [x] **POPUP-06**: Animated stroke order in popup using Hanzi Writer showing how to write the character with play/pause control

### TTS Audio

- [x] **TTS-01**: Azure TTS API route (/api/tts) generates MP3 audio via REST API
- [x] **TTS-02**: TTS supports both Mandarin (zh-CN) and Cantonese (zh-HK) standard neural voices
- [x] **TTS-03**: Audio playback buttons in character popup for both language pronunciations
- [x] **TTS-04**: Sentence read-aloud with configurable speed (slow/medium/fast) via SSML prosody control
- [x] **TTS-05**: Redis caching for TTS responses with TTL based on content length (7 days chars, 24h sentences)

### Vocabulary Saving

- [x] **VOCAB-01**: Save/bookmark button in popup adds word to personal vocabulary list stored in database
- [x] **VOCAB-02**: Saved vocabulary list page showing all bookmarked words with definitions, pronunciation, and date saved

### Integration

- [x] **INTG-01**: Reader accessible from sidebar navigation in Learning section
- [x] **INTG-02**: Lesson integration — reader can open pre-loaded with lesson vocabulary/interaction text

### Cleanup

- [x] **CLEAN-01**: Apply pending DB migrations (0007-0010) and practice table schema to Neon database
- [x] **CLEAN-02**: Commit untracked files (migration 0005, middleware.ts, planning docs)
