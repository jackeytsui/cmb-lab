# Milestones: CantoMando Blueprint

## v6.0 Reading & Dictionary (Shipped: 2026-02-09)

**Delivered:** Chinese text reader with instant character-level dictionary lookup, dual Cantonese/Mandarin pronunciation via Azure TTS, radical/component breakdown, traditional/simplified conversion, vocabulary saving, and lesson integration.

**Phases completed:** 44-49 (16 plans total)

**Key accomplishments:**

- Complete reading interface with text paste, .txt/.pdf import, encoding detection, annotation mode switching (pinyin with tone sandhi, jyutping, or plain), and font size control
- Comprehensive dictionary infrastructure: 145K+ bilingual entries (CC-CEDICT/CC-Canto) and 9.5K character records with radicals, stroke data, and etymology
- Interactive character popup with tone comparison, radical breakdown, animated stroke order (HanziWriter), example words, and vocabulary bookmarking
- Sentence-level TTS read-aloud (Mandarin + Cantonese voices via Azure) with speed control, plus AI-powered tap-to-reveal English translations
- Vocabulary management page for reviewing saved words with search, TTS playback, and lesson-to-reader integration links
- Polish pass with content-shaped loading skeletons, comprehensive error recovery, and accessibility (aria-labels throughout)
