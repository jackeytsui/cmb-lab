**1. Create `src/app/(dashboard)/coach/pronunciation/page.tsx`:**

This is a server component (async function) that queries pronunciation attempts and displays them for coach review.

a. **Access control:** Use `hasMinimumRole("coach")` and `getCurrentUser()` at the top, redirect to /dashboard if not authorized (same pattern as coach/conversations/page.tsx).

b. **Data query:** Query `practiceAttempts` joined with `users` (for student name/email), `practiceSets` (for set title). Filter for completed attempts in the last 30 days. NOTE: We intentionally filter JSONB `pronunciationDetails` in application code rather than using Drizzle's JSONB operators — Drizzle's JSONB query support adds significant complexity (sql`` template literals with JSON path expressions) for marginal DB-level gains on a small result set (limit 100).

   Query pattern:
   ```typescript
   import { eq, desc, isNotNull, gte, and } from "drizzle-orm";

   const thirtyDaysAgo = new Date();
   thirtyDaysAgo.setDate(thirtyDaysAgo.getDate() - 30);

   const recentAttempts = await db
     .select({
       attemptId: practiceAttempts.id,
       userId: practiceAttempts.userId,
       practiceSetId: practiceAttempts.practiceSetId,
       results: practiceAttempts.results,
       score: practiceAttempts.score,
       completedAt: practiceAttempts.completedAt,
       createdAt: practiceAttempts.createdAt,
       studentName: users.name,
       studentEmail: users.email,
       setTitle: practiceSets.title,
     })
     .from(practiceAttempts)
     .innerJoin(users, eq(practiceAttempts.userId, users.id))
     .innerJoin(practiceSets, eq(practiceAttempts.practiceSetId, practiceSets.id))
     .where(
       and(
         isNotNull(practiceAttempts.completedAt),
         gte(practiceAttempts.completedAt, thirtyDaysAgo)
       )
     )
     .orderBy(desc(practiceAttempts.completedAt))
     .limit(100);
   ```

   Then load exercise definitions for all practice sets in the result to get target phrases:
   ```typescript
   // Collect unique practiceSetIds from recentAttempts
   const setIds = [...new Set(recentAttempts.map((a) => a.practiceSetId))];

   // Batch-fetch exercises for those sets
   const exercises = setIds.length > 0
     ? await db
         .select({
           id: practiceExercises.id,
           practiceSetId: practiceExercises.practiceSetId,
           definition: practiceExercises.definition,
         })
         .from(practiceExercises)
         .where(inArray(practiceExercises.practiceSetId, setIds))
     : [];

   // Build lookup: exerciseId -> definition (for targetPhrase)
   const exerciseMap = new Map(exercises.map((e) => [e.id, e.definition]));
   ```

   Then in application code, iterate through results and extract entries that have `pronunciationDetails`, enriching with target phrase from exerciseMap:
   ```typescript
   const pronunciationAttempts = recentAttempts.flatMap((attempt) => {
     const results = attempt.results as Record<string, any> | null;
     if (!results) return [];
     return Object.entries(results)
       .filter(([_, r]) => r?.pronunciationDetails)
       .map(([exerciseId, r]) => {
         const exerciseDef = exerciseMap.get(exerciseId) as any;
         return {
           attemptId: attempt.attemptId,
           exerciseId,
           studentName: attempt.studentName,
           studentEmail: attempt.studentEmail,
           setTitle: attempt.setTitle,
           completedAt: attempt.completedAt,
           targetPhrase: exerciseDef?.targetPhrase || exerciseDef?.sampleAnswer || "(unknown)",
           overallScore: r.pronunciationDetails.overallScore,
           accuracyScore: r.pronunciationDetails.accuracyScore,
           fluencyScore: r.pronunciationDetails.fluencyScore,
           completenessScore: r.pronunciationDetails.completenessScore,
           words: r.pronunciationDetails.words,
           recognizedText: r.pronunciationDetails.recognizedText,
         };
       });
   });
   ```

Based on the vocabulary above, suggest and practice these types of exercises:
