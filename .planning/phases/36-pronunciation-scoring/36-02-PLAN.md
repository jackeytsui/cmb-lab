- `getScoreBgColor(score: number): string` ‚Äî Returns Tailwind bg class: `>=80` -> `bg-green-400/10`, `>=50` -> `bg-yellow-400/10`, `<50` -> `bg-red-400/10`

2. Per-character display ‚Äî `flex flex-wrap justify-center gap-2` with each word from `result.words` as a card:

   - Character text: `text-2xl` with `getScoreColor(word.accuracyScore)`
   - Score number below character: `text-xs text-zinc-500`
   - Rounded-lg padding for touch target
3. Sub-scores grid ‚Äî 3 or 4 columns (Accuracy, Fluency, Completeness, optionally Prosody if present) each showing label + score with color
