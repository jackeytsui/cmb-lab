**SearchBar component (`src/components/search/SearchBar.tsx`):**

Create a `"use client"` component with the following behavior:

1. **State:**
   - `query` (string) -- controlled input value
   - `results` (SearchResult[]) -- from API
   - `isLoading` (boolean) -- fetch in progress
   - `isOpen` (boolean) -- dropdown visible

2. **Debounced search:**
   ```typescript
   import { useDebouncedCallback } from "use-debounce";

   const search = useDebouncedCallback(async (term: string) => {
     if (term.trim().length < 2) {
       setResults([]);
       setIsOpen(false);
       return;
     }
     setIsLoading(true);
     try {
       const res = await fetch(`/api/search?q=${encodeURIComponent(term.trim())}`);
       if (!res.ok) throw new Error("Search failed");
       const data = await res.json();
       setResults(data.results);
       setIsOpen(data.results.length > 0);
     } catch {
       setResults([]);
       setIsOpen(false);
     } finally {
       setIsLoading(false);
     }
   }, 300);
   ```

3. **Click outside to close:**
   Use a `useRef` on the wrapper div and `useEffect` with a mousedown event listener on `document` to detect clicks outside. When clicked outside, set `isOpen(false)`.

4. **Keyboard:** On Escape key, close the dropdown and blur the input.

5. **Render structure:**
   ```
   <div ref={wrapperRef} className="relative">
     <div className="flex items-center gap-2 bg-zinc-800/50 border border-zinc-700 rounded-lg px-3 py-1.5">
       {isLoading ? <Loader2 spinning icon /> : <Search icon />}
       <input
         type="text"
         placeholder="Search courses and lessons..."
         className="bg-transparent text-sm text-white placeholder-zinc-500 outline-none w-48 lg:w-64"
         value={query}
         onChange={(e) => { setQuery(e.target.value); search(e.target.value); }}
         onFocus={() => { if (results.length > 0) setIsOpen(true); }}
         onKeyDown={(e) => { if (e.key === 'Escape') { setIsOpen(false); inputRef.current?.blur(); } }}
       />
     </div>
     {isOpen && <SearchResults results={results} onSelect={() => { setIsOpen(false); setQuery(""); }} />}
   </div>
   ```
   - Import `Search` and `Loader2` from `lucide-react` (already in project)
   - Import `SearchResults` and `SearchResult` type from `./SearchResults`
