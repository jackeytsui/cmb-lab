**2. Create `src/app/(dashboard)/dashboard/listening/ListeningClient.tsx`:**

`"use client"` component that orchestrates the full listening experience. State machine approach:

State variables:
- `videoId: string | null` — current YouTube video ID
- `url: string` — current YouTube URL
- `sessionId: string | null` — videoSession ID from DB
- `captions: Array<{ text: string; startMs: number; endMs: number; sequence: number }> | null`
- `captionStatus: "idle" | "loading" | "success" | "no_captions" | "error"` — tracks extraction state
- `isLoadingVideo: boolean` — for URL submit button state

Flow:
1. **Initial state**: Show only the `UrlInput` component centered on page with a heading "Video Listening Lab" and subheading "Paste a YouTube URL to start practicing"
2. **On URL submit**: Set `isLoadingVideo: true`, set `videoId` from the validated input, call `POST /api/video/extract-captions` with `{ videoId, url }`
3. **On extraction response**:
   - If `captions` exist (success): set `captionStatus: "success"`, store `captions` and `sessionId`
   - If `error: "no_chinese_captions"`: set `captionStatus: "no_captions"`
   - If HTTP error: set `captionStatus: "error"`
   - Always set `isLoadingVideo: false`
4. **After video loaded**: Show split layout:
   - Left side (wider): `YouTubePlayer` with the videoId
   - Right side or below (depending on screen): Caption status + upload area
5. **On successful upload**: Update `captions`, `captionStatus: "success"`, `sessionId`

Layout structure:
```
<div className="flex flex-col gap-6 p-6">
  <div className="flex items-center justify-between">
    <div>
      <h1 className="text-2xl font-bold text-white">Video Listening Lab</h1>
      <p className="text-sm text-zinc-400 mt-1">
        {videoId ? "Playing video" : "Paste a YouTube URL to start practicing"}
      </p>
    </div>
  </div>

  {/* URL Input — always visible at top */}
  <UrlInput onSubmit={handleUrlSubmit} isLoading={isLoadingVideo} />

  {/* Video + Caption area — shown when videoId is set */}
  {videoId && (
    <div className="grid grid-cols-1 lg:grid-cols-3 gap-6">
      <div className="lg:col-span-2">
        <YouTubePlayer videoId={videoId} onReady={handlePlayerReady} />
      </div>
      <div className="space-y-4">
        <CaptionStatus status={captionStatus} captionCount={captions?.length ?? 0} />
        {(captionStatus === "no_captions" || captionStatus === "error" || captionStatus === "success") && sessionId && (
          <CaptionUpload
            videoSessionId={sessionId}
            onUploadComplete={handleUploadComplete}
            hasExistingCaptions={captionStatus === "success"}
          />
        )}
      </div>
    </div>
  )}
</div>
```

Use dark theme styling consistent with the existing app (zinc-900 backgrounds, white/zinc text, cyan accents).
