```typescript

  totalXp: number;
  lessonsCompleted: number;
  practiceCompleted: number;
  daysActive: number;
  goalsMet: number;
}

  thisWeek: WeekStats;
  lastWeek: WeekStats;
  dayOfWeek: number; // 1-7 (Mon-Sun) for goal hit rate calculation
}
```

**Implementation:**

1. Wrap in card container: `bg-zinc-900/50 border border-zinc-800 rounded-lg p-6`.
   - Title "Weekly Summary" in `text-lg font-semibold text-white mb-4`.

2. Create an `AnimatedCount` internal component (reuse the pattern from StreakDisplay):
   ```typescript
   function AnimatedCount({ value }: { value: number }) {
     const [display, setDisplay] = useState(0);
     const ref = useRef<HTMLSpanElement>(null);
     const isInView = useInView(ref, { once: true });
     useEffect(() => {
       if (!isInView || value === 0) { setDisplay(0); return; }
       const controls = animate(0, value, {
         duration: 1.2,
         ease: "easeOut",
         onUpdate(latest) { setDisplay(Math.round(latest)); },
       });
       return () => controls.stop();
     }, [isInView, value]);
     return <span ref={ref}>{display}</span>;
   }
   ```

3. Render 4 stat comparison rows in a 2x2 grid (`grid grid-cols-2 gap-4`):
   - **XP Earned:** AnimatedCount for thisWeek.totalXp, delta vs lastWeek.totalXp
   - **Lessons Completed:** AnimatedCount for thisWeek.lessonsCompleted, delta vs lastWeek
   - **Days Active:** AnimatedCount for thisWeek.daysActive, delta vs lastWeek
   - **Goal Hit Rate:** Percentage = `goalsMet / dayOfWeek * 100` for this week, `goalsMet / 7 * 100` for last week. Show as "X%" with AnimatedCount.

4. Each stat card:
   - Label in `text-xs text-zinc-500 uppercase tracking-wider mb-1`.
   - Value using AnimatedCount in `text-2xl font-bold text-white`.
   - Delta indicator below value:
     - If previous is 0 and current > 0: show "New" in `text-emerald-400 text-xs`.
     - If current > previous: show green up arrow + percentage or absolute change in `text-emerald-400 text-xs`. Use `TrendingUp` icon from lucide.
     - If current < previous: show amber down arrow + change in `text-amber-400 text-xs`. Use `TrendingDown` icon.
     - If equal: show dash "—" in `text-zinc-500 text-xs`.
   - Delta calculation: use absolute difference (not percentage) to avoid division-by-zero. Format as "+12" or "-3".

5. Guard against division by zero everywhere: if dayOfWeek is 0 (shouldn't happen but defensive), treat as 1. If lastWeek values are all 0, show "New" instead of percentages.
