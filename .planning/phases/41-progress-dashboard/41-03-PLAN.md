  <files>src/components/progress/MasteryMap.tsx</files>
  <action>
Create `src/components/progress/MasteryMap.tsx` as a `"use client"` component. (Client for potential future interactivity like expand/collapse, but can work as server component too — mark as client for consistency with other progress components.)

**Props interface:**
```typescript
interface MasteryModule {
  moduleId: string;
  moduleTitle: string;
  totalLessons: number;
  completedLessons: number;
}

interface MasteryCourse {
  courseId: string;
  courseTitle: string;
  modules: MasteryModule[];
  totalLessons: number;
  completedLessons: number;
}

interface MasteryMapProps {
  data: MasteryCourse[];
}
```

**Implementation:**
1. Wrap in card container: `bg-zinc-900/50 border border-zinc-800 rounded-lg p-6`.
   - Title "Mastery" in `text-lg font-semibold text-white mb-4`.

2. For each course, render a collapsible section:
   - Course header: course title, overall completion percentage (e.g., "12/20 lessons - 60%"), and a progress bar.
   - Progress bar: `h-2 rounded-full bg-zinc-800` outer, inner fill `bg-emerald-500` with width as percentage. Use `transition-all duration-500` for smooth fill.
   - Use a disclosure pattern: `useState` to track which courses are expanded (default: all expanded if <= 3 courses, first only if > 3).
   - When expanded, show module list underneath.

3. For each module within an expanded course:
   - Module row: module title (text-sm text-zinc-300), mini completion count (e.g., "3/5"), and a thinner progress bar (h-1).
   - Use `text-zinc-400` for counts, `text-zinc-300` for module title.
   - Progress bar fill: `bg-emerald-500` for started, `bg-emerald-400` when 100%.

4. Empty state: if `data` is empty, show "No courses enrolled yet" in `text-zinc-500 text-center py-8`.

5. Visual indicator for fully completed courses: add a `CheckCircle2` icon (lucide) in `text-emerald-400` next to the course title when 100% complete.

6. Percentage calculation: `Math.round((completedLessons / totalLessons) * 100)`. Guard against totalLessons === 0 (show 0%).
  </action>

    - Division by zero guarded (totalLessons === 0)
    - Empty state renders when data array is empty
  </verify>
