Create `src/hooks/useTTS.ts` as a `"use client"` module. Follow the interface and state patterns from `useLanguagePreference.ts` and the audio ref pattern from `useAudioRecorder.ts`.

**Exported types:**

```typescript
interface TTSOptions {
  language?: "zh-CN" | "zh-HK" | "mandarin" | "cantonese";
  rate?: "x-slow" | "slow" | "medium" | "fast";
  phoneme?: string;
}

interface UseTTSReturn {
  /** Speak the given text. Stops any current playback first. */
  speak: (text: string, options?: TTSOptions) => Promise<void>;
  /** Stop current playback immediately */
  stop: () => void;
  /** True while fetching audio from API */
  isLoading: boolean;
  /** True while audio is actively playing */
  isPlaying: boolean;
  /** Error message from last speak attempt, null if none */
  error: string | null;
}
```

**Implementation details:**

1. **State:** `useState` for `isLoading` (false), `isPlaying` (false), `error` (null).

2. **Refs:**
   - `cacheRef = useRef<Map<string, string>>(new Map())` -- Maps cache keys to blob URLs. Persists across renders within the component lifecycle.
   - `audioRef = useRef<HTMLAudioElement | null>(null)` -- Current playing Audio instance. Only one at a time.
   - `mountedRef = useRef(true)` -- Track mount state to avoid setState on unmounted component.

3. **Cache key format:** `${text}:${language ?? "zh-CN"}:${rate ?? "medium"}:${phoneme ?? ""}` -- Simple string concatenation, no hashing needed on client (the server handles cache key hashing).

4. **`stop` function** (`useCallback`, no deps):
   - If `audioRef.current` exists: call `.pause()`, set `.currentTime = 0`, remove `onended`/`onerror` listeners, set ref to null.
   - Set `isPlaying` to false.

5. **`speak` function** (`useCallback`, deps: `[stop]`):
   - Call `stop()` first to prevent overlap.
   - Set `error` to null.
   - Build cache key from parameters.
   - Check `cacheRef.current.get(key)`:
     - **Cache HIT**: Skip fetch, use existing blob URL.
     - **Cache MISS**:
       - Set `isLoading` to true.
       - `fetch("/api/tts", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text, language: options?.language ?? "zh-CN", rate: options?.rate ?? "medium", phoneme: options?.phoneme }) })`.
       - If `!response.ok`:
         - If 429: set error to `"Too many requests. Please wait a moment."`.
         - If 401: set error to `"Please sign in to use audio."`.
         - Otherwise: set error to `"Failed to generate audio."`.
         - Set `isLoading` to false and return early.
       - `const blob = await response.blob()`.
       - `const url = URL.createObjectURL(blob)`.
       - Store in `cacheRef.current.set(key, url)`.
       - Set `isLoading` to false.
       - Guard: if `!mountedRef.current` return (component unmounted during fetch).
   - Create `const audio = new Audio(url)`.
   - Set `audioRef.current = audio`.
   - Set `isPlaying` to true.
   - Set `audio.onended = () => { if (mountedRef.current) setIsPlaying(false); audioRef.current = null; }`.
   - Set `audio.onerror = () => { if (mountedRef.current) { setIsPlaying(false); setError("Audio playback failed."); } audioRef.current = null; }`.
   - Call `audio.play().catch(err => { ... })` -- catch DOMException for autoplay policy. If autoplay blocked, set error to `"Tap to enable audio playback."` and reset isPlaying.
   - Wrap the entire function body in try/catch. On unexpected error, set error to `"Failed to play audio."`, set isLoading and isPlaying to false.

6. **Cleanup on unmount** (`useEffect`, empty deps):
   - Return cleanup function that:
     - Sets `mountedRef.current = false`.
     - Stops current audio via `stop()`.
     - Iterates `cacheRef.current` values and calls `URL.revokeObjectURL(url)` on each to prevent memory leaks.
     - Clears the cache map.

7. **Return:** `{ speak, stop, isLoading, isPlaying, error }`.

Do NOT fetch on mount -- this is a lazy hook. Audio is only fetched when `speak()` is called.
