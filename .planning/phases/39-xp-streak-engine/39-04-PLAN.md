  <action>
Create `src/components/xp/ActivityRings.tsx` — the Apple Watch-style concentric ring component.

Props interface:
```typescript
interface RingData {
  label: string;     // "Learn" | "Practice" | "Speak"
  current: number;   // today's count (e.g., 1 lesson, 3 exercises, 0 conversations)
  goal: number;      // target (e.g., 1, 5, 1)
  color: string;     // tailwind-compatible hex color
}

interface ActivityRingsProps {
  rings: RingData[];
  size?: number;     // default 140
  className?: string;
}
```

Implementation:
1. SVG with viewBox matching size
2. Three concentric rings, outermost first (Learn = outermost, Practice = middle, Speak = innermost)
3. Each ring has two circles:
   - Background circle: same position, low opacity (0.2), full circumference
   - Progress circle: `motion.circle` from framer-motion with:
     - strokeDasharray = circumference (2 * PI * radius)
     - animate strokeDashoffset from circumference to circumference * (1 - progress)
     - progress = Math.min(current / goal, 1) — cap at 1 for 100% fill
     - strokeLinecap="round" for rounded ends
     - transform rotate(-90) to start from top
     - transition: { duration: 1.5, ease: "easeOut" }
4. Ring layout: strokeWidth = 10, gap between rings = 4
5. Outermost ring radius = center - strokeWidth/2, each inner ring decreases by (strokeWidth + gap)

Color scheme: Learn = emerald (#10b981), Practice = blue (#3b82f6), Speak = amber (#f59e0b)

Below the SVG, render a small legend with colored dots and labels (Learn, Practice, Speak).

Add a `"use client"` directive at the top (framer-motion requires client component).

Respect prefers-reduced-motion: if `window.matchMedia("(prefers-reduced-motion: reduce)")`, set transition duration to 0.

Style with Tailwind classes matching the project's dark theme (bg-zinc-800/900 palette).
  </action>
