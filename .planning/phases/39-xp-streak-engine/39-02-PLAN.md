    Level Progression (linear formula: 100 + (level-1) * 50 per level):
    - getXPForLevel(1) → 100 (XP to go from level 1 to 2)
    - getXPForLevel(2) → 150 (XP to go from level 2 to 3)
    - getXPForLevel(50) → 2550 (XP for last level)
    - getTotalXPForLevel(1) → 0 (total XP to reach level 1)

    - getTotalXPForLevel(3) → 250 (100 + 150)
    - calculateLevel(0) → { level: 1, currentLevelXP: 0, nextLevelXP: 100, totalXP: 0 }
    - calculateLevel(99) → { level: 1, currentLevelXP: 99, nextLevelXP: 100, totalXP: 99 }
    - calculateLevel(100) → { level: 2, currentLevelXP: 0, nextLevelXP: 150, totalXP: 100 }
    - calculateLevel(250) → { level: 3, currentLevelXP: 0, nextLevelXP: 200, totalXP: 250 }
    - calculateLevel(99999) → { level: 50, currentLevelXP: 0, nextLevelXP: 0, totalXP: 99999 } (capped)

    Timezone Date Utilities (using @date-fns/tz TZDate):
    - getTodayInTimezone("UTC") → "YYYY-MM-DD" string for current UTC date
    - getTodayInTimezone("Asia/Hong_Kong") → may differ from UTC by +8 hours
    - getEffectiveDate("UTC") → same as today if hour >= 4, yesterday if hour < 4 (4-hour grace period)
    - areConsecutiveDays("2026-02-06", "2026-02-07") → true
    - areConsecutiveDays("2026-02-06", "2026-02-08") → false
    - areConsecutiveDays("2026-01-31", "2026-02-01") → true (month boundary)

  <implementation>
    Create `src/lib/xp.ts` exporting:

    1. **XP_AMOUNTS** — const object with all XP reward values
    2. **RING_GOALS** — const object with per-dimension daily targets
    3. **DAILY_GOAL_TIERS** — array of { label, value } matching settings page tiers
    4. **getXPForLevel(level: number): number** — XP needed to go FROM level to level+1. Formula: 100 + (level - 1) * 50
    5. **getTotalXPForLevel(level: number): number** — Total XP needed to REACH this level. Sum of arithmetic series.
    6. **calculateLevel(totalXP: number): LevelInfo** — Iterative walk: start at level 1, subtract threshold until insufficient XP. Clamp at 50.
    7. **getTodayInTimezone(timezone: string): string** — Use TZDate.tz(timezone) + format(date, "yyyy-MM-dd")
    8. **getEffectiveDate(timezone: string): string** — If hour < 4 in user's timezone, return yesterday's date string. Otherwise today's.
    9. **areConsecutiveDays(dateA: string, dateB: string): boolean** — Parse as dates, check differenceInCalendarDays === 1

    Also export the LevelInfo type interface.
