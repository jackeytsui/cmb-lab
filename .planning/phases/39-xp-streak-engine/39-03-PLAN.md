  <action>
Create `src/lib/xp-service.ts` with four main functions:

**awardXP(params)** — Core award function:
1. Insert row into xp_events (append-only: userId, source, amount, entityId, entityType)
2. Fetch user's timezone and dailyGoalXp from users table
3. Compute effective date using getEffectiveDate(timezone) from src/lib/xp.ts
4. Upsert daily_activity row for (userId, effectiveDate):
   - ON INSERT: set all counters from this event, set goalXp from user's dailyGoalXp
   - ON CONFLICT (userId, activityDate): increment totalXp, increment the appropriate source counter (lessonCount if lesson_complete, practiceCount if practice_exercise/practice_perfect, conversationCount if voice_conversation), update goalMet if totalXp + amount >= goalXp
5. Check if daily goal JUST met (totalXp crossed threshold): if so, recursively call awardXP with source "daily_goal_met", amount 10 (but do NOT recurse again — check source !== "daily_goal_met" guard)
6. Update users.longestStreak if current streak > stored longestStreak (call getStreak to check)
7. Return { xpAwarded, dailyTotal, goalMet, goalJustMet }

Use `sql` template literals for the ON CONFLICT SET expressions (increment patterns). Match the existing pattern in src/lib/progress.ts for Drizzle upserts.

**getStreak(userId)** — Streak calculation:
1. Fetch user timezone
2. Compute effective today using getEffectiveDate(timezone)
3. Fetch all daily_activity rows for this user, ordered by activityDate DESC
4. Walk backward from effectiveToday:
   - If the most recent day !== effectiveToday, check if it's yesterday (could be mid-day, no activity yet today). If it's > 1 day ago, streak = 0.
   - For each consecutive day: increment streak count
   - On a 1-day gap: if freezesUsed < 2 for the current month AND streakFreezeUsed is false on that gap day, auto-apply freeze (increment streak, count freeze used)
   - On a 2+ day gap: break, streak ends
5. Count freezesUsedThisMonth from daily_activity rows where streakFreezeUsed = true and activityDate is in current month (user's timezone)
6. Fetch longestStreak from users table
7. Return { currentStreak, longestStreak, freezesUsedThisMonth, freezesRemaining: 2 - freezesUsedThisMonth }

**getDailyActivity(userId)** — Today's summary:
1. Fetch user timezone and dailyGoalXp
2. Compute effective today
3. Query daily_activity for (userId, effectiveDate)
4. Return the row (or default zeros if no row exists yet today)

**getXPDashboard(userId)** — Aggregated dashboard response:
1. Get total XP: SUM(amount) from xp_events WHERE userId
2. Call calculateLevel(totalXP) for level info
3. Call getStreak(userId) for streak info
4. Call getDailyActivity(userId) for today's summary
5. Compute ring data from daily activity: { learn: lessonCount/RING_GOALS.learn, practice: practiceCount/RING_GOALS.practice, speak: conversationCount/RING_GOALS.speak }
6. Return combined object with level, streak, daily, rings

Import pure functions from src/lib/xp.ts. Import schema from src/db/schema. Use Drizzle db from src/db.
