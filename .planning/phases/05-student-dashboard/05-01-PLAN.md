1. **Update the query** to aggregate progress data:
   - Join `lessonProgress` table
   - Count total lessons per course (via modules -> lessons joins)
   - Count completed lessons (where completedAt is not null)
   - Use Drizzle's `sql` template for COUNT aggregations

**Query structure:**
```typescript
const userCourses = await db
  .select({
    id: courses.id,
    title: courses.title,
    description: courses.description,
    thumbnailUrl: courses.thumbnailUrl,
    accessTier: courseAccess.accessTier,
    totalLessons: sql<number>`COUNT(DISTINCT ${lessons.id})`.as('total_lessons'),
    completedLessons: sql<number>`COUNT(DISTINCT CASE WHEN ${lessonProgress.completedAt} IS NOT NULL THEN ${lessonProgress.lessonId} END)`.as('completed_lessons'),
  })
  .from(courseAccess)
  .innerJoin(users, eq(courseAccess.userId, users.id))
  .innerJoin(courses, eq(courseAccess.courseId, courses.id))
  .leftJoin(modules, eq(modules.courseId, courses.id))
  .leftJoin(lessons, eq(lessons.moduleId, modules.id))
  .leftJoin(lessonProgress, and(
    eq(lessonProgress.lessonId, lessons.id),
    eq(lessonProgress.userId, users.id)
  ))
  .where(
    and(
      eq(users.clerkId, clerkId),
      isNull(courses.deletedAt),
      or(
        isNull(courseAccess.expiresAt),
        gt(courseAccess.expiresAt, new Date())
      )
    )
  )
  .groupBy(courses.id, courses.title, courses.description, courses.thumbnailUrl, courseAccess.accessTier);
```
