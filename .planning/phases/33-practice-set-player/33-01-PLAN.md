Create `src/lib/practice-grading.ts` with these exports:

```typescript
export interface GradeResult {
  isCorrect: boolean;
  score: number; // 0-100
  feedback: string;
  explanation?: string;
}
```

**gradeMultipleChoice(selectedOptionId: string, definition: MultipleChoiceDefinition): GradeResult**
- Compare selectedOptionId against definition.correctOptionId
- On correct: score=100, feedback="Correct!"
- On wrong: score=0, feedback includes the correct option text (find by correctOptionId in definition.options)
- Always include definition.explanation if present

**gradeFillInBlank(answers: string[], definition: FillInBlankDefinition): GradeResult**
- For each blank, compare student answer against correctAnswer AND acceptableAnswers
- Normalize both sides with `.trim().toLowerCase()` before comparison (CRITICAL for Chinese text)
- Score = Math.round((correctCount / totalBlanks) * 100)
- isCorrect only when ALL blanks correct (score === 100)
- Feedback: "All blanks correct!" or "X/Y blanks correct"

**gradeMatching(userPairs: { leftId: string; rightId: string }[], definition: MatchingDefinition): GradeResult**
- Each definition.pair has an `id` field. The left column shows `pair.left`, the right column shows `pair.right`.
- The correct mapping is: leftId === pair.id AND rightId === pair.id (same pair matched)
- Count how many userPairs have leftId === rightId (meaning student connected the same pair's left and right)
- Score = Math.round((correctCount / definition.pairs.length) * 100)

**gradeOrdering(orderedItemIds: string[], definition: OrderingDefinition): GradeResult**
- The student submits an array of item IDs in their chosen order
- For each item at index i in orderedItemIds, find the item in definition.items and check if item.correctPosition === i
- Score = Math.round((correctCount / definition.items.length) * 100)
