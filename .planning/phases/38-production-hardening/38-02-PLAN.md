  <name>Task 1: Add indexes to core content and progress schema files</name>
  <files>src/db/schema/courses.ts, src/db/schema/access.ts, src/db/schema/interactions.ts, src/db/schema/progress.ts, src/db/schema/submissions.ts, src/db/schema/notes.ts, src/db/schema/conversations.ts</files>
  <action>
  Follow the existing pattern from `notifications.ts` — add `index` to the import from `drizzle-orm/pg-core` and add a third argument callback to each `pgTable()` call that returns an array of index declarations.

  **Pattern:**
  ```typescript
  import { ..., index } from "drizzle-orm/pg-core";

  export const tableName = pgTable("table_name", {
    // columns...
  }, (table) => [
    index("table_name_column_idx").on(table.columnName),
  ]);
  ```

  **Naming convention:** `{table_name}_{column_name}_idx`

  **Indexes to add per file:**

  **courses.ts:**
  - `modules`: `index("modules_course_id_idx").on(table.courseId)`
  - `lessons`: `index("lessons_module_id_idx").on(table.moduleId)`

  **access.ts:**
  - `courseAccess`: `index("course_access_user_id_idx").on(table.userId)`, `index("course_access_course_id_idx").on(table.courseId)`

  **interactions.ts:**
  - `interactions`: `index("interactions_lesson_id_idx").on(table.lessonId)`
  - `interactionAttempts`: `index("interaction_attempts_interaction_id_idx").on(table.interactionId)`, `index("interaction_attempts_user_id_idx").on(table.userId)`

  **progress.ts:**
  - `lessonProgress`: SKIP userId (covered by unique(userId, lessonId) — leading column). Add `index("lesson_progress_lesson_id_idx").on(table.lessonId)` only.

  **submissions.ts:**
  - `submissions`: `index("submissions_user_id_idx").on(table.userId)`, `index("submissions_interaction_id_idx").on(table.interactionId)`, `index("submissions_lesson_id_idx").on(table.lessonId)`, `index("submissions_reviewed_by_idx").on(table.reviewedBy)`

  **notes.ts:**
  - `coachFeedback`: SKIP submissionId (has `.unique()` — implicit index). Add `index("coach_feedback_coach_id_idx").on(table.coachId)`
  - `coachNotes`: `index("coach_notes_coach_id_idx").on(table.coachId)`, `index("coach_notes_student_id_idx").on(table.studentId)`, `index("coach_notes_submission_id_idx").on(table.submissionId)`

  **conversations.ts:**
  - `conversations`: `index("conversations_user_id_idx").on(table.userId)`, `index("conversations_lesson_id_idx").on(table.lessonId)`
  - `conversationTurns`: `index("conversation_turns_conversation_id_idx").on(table.conversationId)`

  **IMPORTANT:** If a table already has a third argument (existing constraints or indexes), add the new indexes to the existing return array instead of replacing it. Read each file carefully before editing.
  </action>
  <verify>For each file, grep for `index(` to confirm indexes were added. Count total: courses.ts (2), access.ts (2), interactions.ts (3), progress.ts (1), submissions.ts (4), notes.ts (4), conversations.ts (3) = 19 indexes.</verify>
