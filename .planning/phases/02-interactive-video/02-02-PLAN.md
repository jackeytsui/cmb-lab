<objective>
Create the overlay composition system and subtitle rendering with Chinese/Pinyin/Jyutping annotations.

Purpose: The overlay provides the visual container for interactions (input fields, prompts) that appears when video pauses. Subtitles with Ruby annotations enable simultaneous display of Chinese characters with their romanization (Pinyin for Mandarin, Jyutping for Cantonese), which is essential for language learners.

Output: Complete interactive video system with animated overlays and annotated subtitles, ready for interaction content in Phase 3.
</objective>

  <action>
    **Part A: Create `src/hooks/useSubtitlePreference.ts`**

    Hook for managing annotation visibility preference:

    1. State: `showPinyin: boolean`, `showJyutping: boolean`
    2. Load preference from localStorage on mount (key: 'subtitle-prefs')
    3. Default: both true on first visit
    4. Persist changes to localStorage
    5. Return: { showPinyin, showJyutping, togglePinyin, toggleJyutping, setShowPinyin, setShowJyutping }

    **Part B: Create `src/components/video/SubtitleOverlay.tsx`**

    Props:
    - `currentTime: number` - current video playback time
    - `cues: SubtitleCue[]` - array of subtitle cues with timestamps
    - `showPinyin: boolean` - whether to show Pinyin annotations
    - `showJyutping: boolean` - whether to show Jyutping annotations

    **Rendering (from RESEARCH.md Pattern 5):**
    1. Find active cue where startTime <= currentTime < endTime
    2. If no active cue, render nothing
    3. Parse Chinese characters and annotations:
       - Assume parallel arrays: chars[i] pairs with pinyin[i] and jyutping[i]
       - Handle cases where pinyin/jyutping arrays are shorter
    4. Use HTML `<ruby>` and `<rt>` elements for annotations
    5. Pinyin in yellow-400 color, Jyutping in cyan-400 color
    6. Wrap in semi-transparent dark background for readability

    **Positioning:**
    - Absolute position at bottom of video container
    - Leave space for Mux controls (bottom-16 or similar)
    - Centered horizontally

    **Styling:**
    - Base text: text-2xl or text-xl, white
    - Annotation text: text-sm
    - Use `ruby-align: center` CSS for proper centering
    - Add letter-spacing to base text for visual balance
  </action>
