**Create src/lib/sentences.ts:**

1. Define `SentenceRange` interface:
   ```tsx
   export interface SentenceRange {
     /** Start index in the segments array */
     startIndex: number;
     /** End index (inclusive) in the segments array */
     endIndex: number;
     /** Combined text of all segments in this sentence */
     text: string;
   }
   ```

2. Define sentence terminator regex including both Chinese and Western punctuation:
   ```tsx
   const SENTENCE_TERMINATORS = /[。！？；.!?]/;
   ```

3. Implement `detectSentences(segments: WordSegment[]): SentenceRange[]`:
   - Walk through segments array
   - When a segment's text matches SENTENCE_TERMINATORS, end the current sentence (inclusive of the terminator)
   - Start a new sentence at the next segment
   - Handle trailing text without a terminator as its own sentence
   - If a single resulting "sentence" is longer than 100 characters (text property), split it at the nearest whitespace/punctuation segment boundary to stay within TTS character limits
   - Skip pure-whitespace sentences (trim check)
   - Import `WordSegment` from `@/lib/segmenter`

**Create src/app/api/reader/translate/route.ts:**

1. Auth check: `const { userId } = await auth()` — return 401 if not authenticated.

2. Parse body: `{ text: string }`. Validate:
   - `text` is a non-empty string
   - `text.length <= 500` (TTS limit applies here too — prevent abuse)
   - Return 400 with error message on validation failure

3. Call OpenAI via the AI SDK (`generateText` from `ai` package, `openai` from `@ai-sdk/openai`):
   ```tsx
   const { text: translation } = await generateText({
     model: openai("gpt-4o-mini"),
     system: "Translate the following Chinese text to natural, fluent English. Return ONLY the English translation, nothing else. Do not include the original Chinese text.",
     prompt: text,
   });
   ```

4. Return `{ translation }` as JSON response.

5. Wrap in try/catch — on error, return 500 with `{ error: "Translation failed" }`.

6. Imports needed: `NextRequest`, `NextResponse` from `next/server`, `auth` from `@clerk/nextjs/server`, `generateText` from `ai`, `openai` from `@ai-sdk/openai`.
