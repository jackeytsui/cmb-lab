   - `completedAt`: timestamp("completed_at") (nullable - set when both video + interactions done)

2. **upsertLessonProgress function:**
   - Signature: `async function upsertLessonProgress(input: { userId: string; lessonId: string; videoWatchedPercent?: number; interactionCompleted?: boolean; }): Promise<LessonProgress>`
   - Use Drizzle's `onConflictDoUpdate` targeting the composite unique constraint
   - For videoWatchedPercent: use SQL `GREATEST()` to only increase, never decrease
   - If videoWatchedPercent >= 95: set videoCompletedAt using `COALESCE(video_completed_at, NOW())`
   - If interactionCompleted: increment interactionsCompleted by 1
   - Always update lastAccessedAt to new Date()
   - Return the upserted record

3. **checkLessonCompletion function:**
   - Signature: `async function checkLessonCompletion(userId: string, lessonId: string, languagePreference: LanguagePreference): Promise<CompletionStatus>`
   - Query lesson_progress for current state (may not exist yet)

   - Filter interactions using `filterInteractionsByPreference` from @/lib/interactions
   - Query interactionAttempts for this user where isCorrect = true
   - Count how many of the filtered interactions have at least one correct attempt
   - Video is complete if videoCompletedAt is not null
   - Return CompletionStatus with all computed values
