1. Create `src/components/admin/KbEntryForm.tsx` ("use client"):

Props interface:
```typescript
interface KbEntryFormProps {
  mode: "create" | "edit";
  entry?: { id: string; title: string; content: string; categoryId: string | null; status: string };
  categories: { id: string; name: string }[];
  onSuccess?: () => void;
}
```

Form fields:
- Title: text input (required)
- Category: select dropdown with "No category" option + all categories
- Status: select dropdown with "Published" and "Draft" options
- Content: textarea (required) — large textarea (min-h-[300px]) for rich content
  - Placeholder: "Enter knowledge content... Supports markdown formatting."

Behavior:
- Use React useState for form fields (controlled inputs)
- On submit:
  - If mode === "create": POST to /api/admin/knowledge/entries with { title, content, categoryId, status }
  - If mode === "edit": PATCH to /api/admin/knowledge/entries/[entry.id] with changed fields
  - Show loading state on submit button
  - On success: redirect to /admin/knowledge (use router.push) or call onSuccess
  - On error: show error message in red text below form
- Delete button (edit mode only): DELETE /api/admin/knowledge/entries/[entry.id]
  - Confirm with window.confirm("Are you sure?")
  - On success: redirect to /admin/knowledge

Style: Use existing admin form patterns (bg-zinc-800 border-zinc-700 rounded-lg, label text-zinc-300, input bg-zinc-900 border-zinc-700 text-white). Import Button from @/components/ui/button.
