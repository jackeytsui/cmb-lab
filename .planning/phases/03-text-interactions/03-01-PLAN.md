   - id: uuid primary key with defaultRandom()
   - lessonId: uuid FK to lessons.id with cascade delete
   - timestamp: integer (seconds into video)
   - type: interactionTypeEnum
   - language: interactionLanguageEnum (which language this tests)
   - prompt: text (what to show the student)
   - expectedAnswer: text (for AI grading context, nullable)
   - correctThreshold: integer default 80 (score 0-100 needed to pass)
   - sortOrder: integer default 0
   - createdAt, updatedAt timestamps
   - deletedAt nullable timestamp

4. interactionAttempts table (for tracking student attempts):
   - id: uuid primary key
   - interactionId: uuid FK to interactions.id with cascade delete
   - userId: uuid FK to users.id with cascade delete
   - response: text (what student submitted)
   - score: integer (0-100 from AI)
   - isCorrect: boolean
   - feedback: text (AI feedback)
   - attemptNumber: integer
   - createdAt timestamp

5. Add relations:
   - interactions -> lesson (one)
   - interactionAttempts -> interaction (one)
   - interactionAttempts -> user (one)

6. Export types: Interaction, NewInteraction, InteractionAttempt, NewInteractionAttempt

```typescript
import { z } from "zod";

export const textInteractionSchema = z.object({
  response: z
    .string()
    .min(1, { message: "Please enter your response" })
    .max(500, { message: "Response too long (max 500 characters)" }),
});

export type TextInteractionFormValues = z.infer<typeof textInteractionSchema>;
```
