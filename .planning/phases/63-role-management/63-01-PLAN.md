  <name>Task 1: Create role CRUD service module</name>
  <files>src/lib/roles.ts</files>
  <action>
Create `src/lib/roles.ts` following the `src/lib/tags.ts` pattern. Export these functions:

1. **getRoles(filters?: { search?: string })**: Query `roles` table WHERE `deletedAt IS NULL`. If `search` is provided, trim it first (`search.trim()`); if the trimmed value is empty/falsy, ignore it and return all non-deleted roles. If the trimmed value is non-empty, filter by `ilike(roles.name, '%trimmedSearch%')`. Order by `asc(roles.sortOrder), asc(roles.name)`. Also query `userRoles` table grouped by `roleId` to get student counts. Return `(Role & { studentCount: number })[]`.

2. **getRoleById(roleId: string)**: Get a single role WHERE `id = roleId AND deletedAt IS NULL`. Return `Role | null`.

3. **createRole(data: { name, description?, color, createdBy? })**: Auto-increment `sortOrder` by querying max sortOrder of non-deleted roles. Insert into `roles` table with `name.trim()`, `description?.trim() || null`, color, computed sortOrder, and createdBy. Return the created `Role`.

4. **updateRole(roleId, data: { name?, description?, color?, sortOrder? })**: Update `roles` WHERE `id = roleId AND deletedAt IS NULL`. Trim string values. Return `Role | null`.

5. **softDeleteRole(roleId: string)**: First count `userRoles` rows WHERE `roleId = roleId`. If count > 0, return `{ deleted: false, reason: "Cannot delete: X student(s) are currently assigned this role" }`. If count === 0, update the role to set `deletedAt = new Date()` AND rename the role to `${role.name}_deleted_${Date.now()}` to free the unique name constraint. Return `{ deleted: boolean, reason?: string }`.

Import from: `@/db`, `@/db/schema` (roles, userRoles), `drizzle-orm` (eq, and, isNull, ilike, asc, count, sql, max).

Important: The `name` column has a UNIQUE constraint. Soft-delete must rename to free the name (Pitfall 3 from research). Use pattern: append `_deleted_${Date.now()}` to the name when soft-deleting.
  </action>

**File 2: `src/app/api/admin/roles/[roleId]/route.ts`**
