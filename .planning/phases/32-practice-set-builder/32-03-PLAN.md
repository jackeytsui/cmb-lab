    Create `src/components/admin/builder/BuilderCanvas.tsx` — "use client" component.

    This is the main canvas area where exercises are displayed as sortable blocks. It wraps each ExerciseBlock in a sortable container.

    **Props:**
    ```typescript
    interface BuilderCanvasProps {
      exercises: BuilderExercise[];
      editingId: string | null;
      onEdit: (id: string) => void;
      onRemove: (id: string) => void;
      renderEditor: (exercise: BuilderExercise) => React.ReactNode;
    }
    ```

    **SortableExerciseItem wrapper:**
    ```tsx
    function SortableExerciseItem({
      exercise,
      index,
      isEditing,
      onEdit,
      onRemove,
      renderEditor,
    }: SortableItemProps) {
      const { ref, isDragging } = useSortable({
        id: exercise.id,
        index,
        type: 'exercise',
        group: 'canvas',
      });

      return (
        <div
          ref={ref}
          data-dragging={isDragging}
          className={cn(
            "transition-all",
            isDragging && "opacity-50 scale-[0.98]",
          )}
        >
          {isEditing ? (
            renderEditor(exercise)
          ) : (
            <ExerciseBlock
              exercise={exercise}
              index={index}
              onEdit={onEdit}
              onRemove={onRemove}
            />
          )}
        </div>
      );
    }
    ```

    **BuilderCanvas component:**
    - Renders a vertical list of SortableExerciseItem components
    - Has a min-height to act as a drop zone even when empty
    - Empty state: "Drag exercise types from the palette to add questions" with a dashed border
    - Uses `useDroppable` to make the canvas a valid drop target for palette items:
      ```tsx
      const { ref: canvasRef } = useDroppable({
        id: 'canvas-drop-zone',
        type: 'canvas',
      });
      ```
    - Wraps the list in the droppable ref
    - Shows exercise count in a subtle header: "Questions ({n})"

    **CRITICAL: @dnd-kit/react import paths:**
    - `import { useSortable } from '@dnd-kit/react/sortable';`
    - `import { useDroppable } from '@dnd-kit/react';`
    - NOT from @dnd-kit/core or @dnd-kit/sortable (legacy packages)

    The DragDropProvider and onDragEnd/onDragOver handlers are NOT in this component — they go in the parent BuilderClient (Plan 05). This component only renders the sortable items and provides refs.
