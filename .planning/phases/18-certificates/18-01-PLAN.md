  <action>
    1. Install dependencies:
       ```bash
       npm install @react-pdf/renderer nanoid
       ```

    2. Download font files to `public/fonts/` directory:
       - NotoSansSC-Regular.ttf and NotoSansSC-Bold.ttf from Google Fonts notofonts releases (Subset OTF, ~5-9MB each)
       - Inter-Regular.ttf and Inter-Bold.ttf from Google Fonts
       - Use curl to download directly from Google Fonts CDN or GitHub releases

    3. Create `src/db/schema/certificates.ts`:
       - Table `certificates` with columns:
         - `id`: uuid, defaultRandom, primaryKey
         - `userId`: uuid, notNull, references users.id (onDelete cascade)
         - `courseId`: uuid, notNull, references courses.id (onDelete cascade)
         - `verificationId`: text, notNull, unique -- generated via nanoid(12) for short URL-safe IDs
         - `studentName`: text, notNull -- snapshot at time of completion (so name changes don't affect cert)
         - `courseTitle`: text, notNull -- snapshot at time of completion
         - `completedAt`: timestamp, notNull -- when course was completed
         - `createdAt`: timestamp, notNull, defaultNow
       - UNIQUE constraint on (userId, courseId) -- one certificate per student per course
       - Relations: certificate belongs to user and course

    4. Update `src/db/schema/index.ts`:
       - Add `export * from "./certificates";` at the end

    5. Run `npx drizzle-kit push` to apply the schema migration.
  </action>
  <verify>
    - `npm ls @react-pdf/renderer nanoid` shows both installed
    - Font files exist in `public/fonts/` (ls confirms 4 .ttf files)
    - `npx drizzle-kit push` completes without errors
    - `npm run build` passes (no type errors in schema)
  </verify>

<task type="auto">
  <name>Task 2: Certificate library and PDF template component</name>
  <files>
    src/lib/certificates.ts
    src/components/certificate/CertificateDocument.tsx
  </files>
  <action>
    1. Create `src/lib/certificates.ts` with these functions:

       **checkCourseCompletion(userId: string, courseId: string): Promise<boolean>**
       - Query all non-deleted lessons in the course (join modules -> lessons, filter deletedAt IS NULL for both)
       - Query lessonProgress for this user where completedAt IS NOT NULL
       - Return true only if ALL lessons have completedAt set

       **createCertificate(userId: string, courseId: string): Promise<Certificate>**
       - First call checkCourseCompletion -- throw if not complete
       - Look up user name from users table, course title from courses table
       - Generate verificationId via `nanoid(12)`
       - Insert into certificates table with ON CONFLICT (userId, courseId) DO NOTHING for idempotency
       - If conflict (cert already exists), return the existing certificate
       - Return the certificate record

       **getCertificateByVerificationId(verificationId: string): Promise<Certificate | null>**
       - Simple SELECT by verificationId

       **getCertificatesForUser(userId: string): Promise<Certificate[]>**
       - SELECT all certificates for this userId, ordered by completedAt DESC

    2. Create `src/components/certificate/CertificateDocument.tsx`:
       - Use `@react-pdf/renderer` with `Document`, `Page`, `View`, `Text`, `Font`
       - Register fonts at module scope using `Font.register()`:
         - Inter-Regular and Inter-Bold for Latin text
         - NotoSansSC-Regular and NotoSansSC-Bold for Chinese characters
         - Use `path.join(process.cwd(), 'public/fonts/...')` for font paths (NOT relative paths)
       - Certificate layout (landscape A4):
         - Decorative border (double line or gradient effect using nested View boxes)
         - Header: "Certificate of Completion" in large Inter Bold
         - Body: "This certifies that" line
         - Student name: Large text, use fontFamily that includes NotoSansSC for Chinese fallback
         - "has successfully completed" line
         - Course title: Emphasized text with NotoSansSC support
         - Completion date: formatted as "January 30, 2026"
         - Verification ID: Small text at bottom, "Verify at: [domain]/verify/[id]"
       - Props interface: `{ studentName: string; courseTitle: string; completedAt: Date; verificationId: string }`
       - Export as named export `CertificateDocument`

       IMPORTANT: For Chinese character support, register the NotoSansSC font family and use a composite fontFamily approach. The @react-pdf/renderer Font.register supports multiple font families. Use NotoSansSC as the font family for student name and course title (since these may contain Chinese). Use Inter for other text.
  </action>
