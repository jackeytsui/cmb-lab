<task type="auto">
  <name>Task 1: Create ChineseAnnotation component and update ChatMessage to parse annotations</name>
  <files>src/components/chat/ChineseAnnotation.tsx, src/components/chat/ChatMessage.tsx</files>
  <action>
    **Create `src/components/chat/ChineseAnnotation.tsx`** as a client component:
    - Export `ChineseAnnotation` component
    - Props: `text: string` — a single annotated segment in format `[char|pinyin|jyutping]` or `[char|pinyin]`
    - Parse the text: split by `|` after removing brackets
    - Render as `<ruby>` element:
      - Chinese character(s) as the base text
      - First annotation (Pinyin) as `<rt className="text-xs text-yellow-400 font-normal">`
      - Second annotation (Jyutping) if present as `<rt className="text-xs text-cyan-400 font-normal">`
    - Use yellow for Pinyin, cyan for Jyutping (matches existing SubtitleOverlay color convention)

    **Create a `parseAnnotatedText` helper** (in same file or ChatMessage):
    - Export `parseAnnotatedText(text: string): Array<{ type: 'text' | 'annotation'; content: string }>`
    - Use regex to find `\[([^\]]+)\]` patterns in the text
    - Split text into alternating plain text and annotation segments
    - Plain text segments: `{ type: 'text', content: '...' }`
    - Annotation segments: `{ type: 'annotation', content: 'char|pinyin|jyutping' }`

    **Update `src/components/chat/ChatMessage.tsx`:**
    - Import `ChineseAnnotation` and `parseAnnotatedText`
    - When rendering text parts, call `parseAnnotatedText(part.text)` on each text part
    - For each segment:
      - `type: 'text'` — render as plain `<span>`
      - `type: 'annotation'` — render as `<ChineseAnnotation text={segment.content} />`
    - Wrap the parsed output in the existing `<p className="whitespace-pre-wrap">` container

    **Annotation format for the LLM:**
    The system prompt will instruct the LLM to use `[character|pinyin|jyutping]` format. Example:
    - Input from LLM: `"Hello! In Mandarin, 'hello' is [你好|nǐ hǎo|nei5 hou2]."`
    - Rendered: "Hello! In Mandarin, 'hello' is " + ruby-annotated 你好 with Pinyin above and Jyutping below

    **Edge cases to handle:**
    - Text with no annotations: renders as plain text (no brackets found)
    - Mixed text: some annotated, some not — alternating segments
    - Malformed brackets (no pipe): treat as plain text, don't crash
  </action>
  <verify>
    `npx tsc --noEmit` passes.
    `parseAnnotatedText('Hello [你好|nǐ hǎo|nei5 hou2] world')` returns 3 segments: text, annotation, text.
    `parseAnnotatedText('No annotations here')` returns 1 text segment.
  </verify>
  <done>Chinese annotations render with ruby elements in chat messages. Pinyin in yellow, Jyutping in cyan. Plain text renders normally.</done>
