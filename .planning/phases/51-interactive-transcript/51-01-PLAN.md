**TranscriptLine.tsx** -- Create a new "use client" component at `src/components/video/TranscriptLine.tsx`. Props:
- `ref?: React.Ref<HTMLDivElement>` (React 19 native ref prop, no forwardRef needed)
- `isActive: boolean`
- `text: string`
- `startMs: number`
- `onClick: () => void`

Render a `<div>` with:
- `ref={ref}`, `role="button"`, `tabIndex={0}`
- `onClick={onClick}`, `onKeyDown` handler that calls `onClick` on Enter or Space (with `e.preventDefault()`)
- Tailwind classes: `px-3 py-2 rounded-md cursor-pointer transition-colors text-sm hover:bg-zinc-800`

- A timestamp span: `<span className="text-xs text-zinc-600 mr-2 tabular-nums">{formatTimestamp(startMs)}</span>`
- Then the `{text}` content

Include a local `formatTimestamp(ms: number): string` helper that converts milliseconds to `M:SS` format (e.g., 65000 -> "1:05"). Floor to seconds, compute minutes and zero-padded seconds.

Export the component as a named export: `export function TranscriptLine(...)`.

**TranscriptPanel.tsx** -- Create a new "use client" component at `src/components/video/TranscriptPanel.tsx`. Props:
- `captions: { text: string; startMs: number; endMs: number; sequence: number }[]`
- `activeCaptionIndex: number` (default -1, no line active)
- `onLineClick: (index: number) => void`
- `scrollContainerRef?: React.Ref<HTMLDivElement>` (for parent to attach scroll listener)
- `activeLineRef?: React.Ref<HTMLDivElement>` (for parent to trigger scrollIntoView)

Render structure:
```
<div className="flex flex-col h-full">
  <div className="flex items-center justify-between px-4 py-3 border-b border-zinc-800">
    <h2 className="text-sm font-semibold text-zinc-300">Transcript</h2>
    <span className="text-xs text-zinc-500">{captions.length} lines</span>
  </div>
  <div
    ref={scrollContainerRef}
    className="flex-1 overflow-y-auto p-2 space-y-1"
  >
    {captions.length === 0 ? (
      <div className="flex flex-col items-center justify-center h-full text-zinc-500 text-sm">
        <p>No transcript available</p>
        <p className="text-xs mt-1">Extract or upload captions to see the transcript</p>
      </div>
    ) : (
      captions.map((caption, index) => (
        <TranscriptLine
          key={caption.sequence}
          ref={index === activeCaptionIndex ? activeLineRef : undefined}
          isActive={index === activeCaptionIndex}
          text={caption.text}
          startMs={caption.startMs}
          onClick={() => onLineClick(index)}
        />
      ))
    )}
  </div>
</div>
```

Import TranscriptLine from `@/components/video/TranscriptLine`.
