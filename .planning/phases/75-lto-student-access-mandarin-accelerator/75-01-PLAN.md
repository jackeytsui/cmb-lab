1. Create `src/db/schema/accelerator.ts` with the following tables (follow the exact Drizzle patterns used in existing schema files like grammar.ts):

**typingLanguageEnum:**
```typescript
export const typingLanguageEnum = pgEnum("typing_language", ["mandarin", "cantonese"]);
```

**typingSentences table:**
- `id`: uuid, defaultRandom, primaryKey
- `language`: typingLanguageEnum, notNull
- `sortOrder`: integer("sort_order"), notNull, default(0)
- `chineseText`: text("chinese_text"), notNull (the expected answer)
- `englishText`: text("english_text"), notNull (English translation prompt)
- `romanisation`: text("romanisation"), notNull (pinyin for Mandarin, jyutping for Cantonese)
- `createdBy`: uuid("created_by"), references users.id
- `createdAt`: timestamp("created_at"), notNull, defaultNow()
- `updatedAt`: timestamp("updated_at"), notNull, defaultNow(), $onUpdate(() => new Date())
- Indexes: on `language`, on `sortOrder`

**typingProgress table:**
- `id`: uuid, defaultRandom, primaryKey
- `userId`: uuid("user_id"), notNull, references users.id with onDelete cascade
- `sentenceId`: uuid("sentence_id"), notNull, references typingSentences.id with onDelete cascade
- `completedAt`: timestamp("completed_at"), notNull, defaultNow()
- Unique index on (userId, sentenceId)
- Index on userId

**conversationScripts table:**
- `id`: uuid, defaultRandom, primaryKey
- `title`: text("title"), notNull
- `description`: text("description")
- `speakerRole`: text("speaker_role"), notNull (e.g. "Waiter")
- `responderRole`: text("responder_role"), notNull (e.g. "Customer")
- `sortOrder`: integer("sort_order"), notNull, default(0)
- `createdBy`: uuid("created_by"), references users.id
- `createdAt`: timestamp("created_at"), notNull, defaultNow()
- `updatedAt`: timestamp("updated_at"), notNull, defaultNow(), $onUpdate(() => new Date())

**scriptLines table:**
- `id`: uuid, defaultRandom, primaryKey
- `scriptId`: uuid("script_id"), notNull, references conversationScripts.id with onDelete cascade
- `sortOrder`: integer("sort_order"), notNull, default(0)
- `role`: text("role"), notNull ("speaker" or "responder")
- `cantoneseText`: text("cantonese_text"), notNull
- `mandarinText`: text("mandarin_text"), notNull
- `cantoneseRomanisation`: text("cantonese_romanisation"), notNull (jyutping)
- `mandarinRomanisation`: text("mandarin_romanisation"), notNull (pinyin)
- `englishText`: text("english_text"), notNull
- `cantoneseAudioUrl`: text("cantonese_audio_url") (nullable, Vercel Blob URL)
- `mandarinAudioUrl`: text("mandarin_audio_url") (nullable, Vercel Blob URL)
- `createdAt`: timestamp("created_at"), notNull, defaultNow()
- Indexes: on scriptId, on sortOrder

**scriptLineProgress table:**
- `id`: uuid, defaultRandom, primaryKey
- `userId`: uuid("user_id"), notNull, references users.id with onDelete cascade
- `lineId`: uuid("line_id"), notNull, references scriptLines.id with onDelete cascade
- `selfRating`: text("self_rating"), notNull ("good" or "not_good")
- `updatedAt`: timestamp("updated_at"), notNull, defaultNow(), $onUpdate(() => new Date())
- Unique index on (userId, lineId)
- Index on userId

**curatedPassages table:**
- `id`: uuid, defaultRandom, primaryKey
- `title`: text("title"), notNull
- `body`: text("body"), notNull (Chinese text content)
- `sortOrder`: integer("sort_order"), notNull, default(0)
- `createdBy`: uuid("created_by"), references users.id
- `createdAt`: timestamp("created_at"), notNull, defaultNow()
- `updatedAt`: timestamp("updated_at"), notNull, defaultNow(), $onUpdate(() => new Date())

**passageReadStatus table:**
- `id`: uuid, defaultRandom, primaryKey
- `userId`: uuid("user_id"), notNull, references users.id with onDelete cascade
- `passageId`: uuid("passage_id"), notNull, references curatedPassages.id with onDelete cascade
- `readAt`: timestamp("read_at"), notNull, defaultNow()
- Unique index on (userId, passageId)
- Index on userId

Add Drizzle `relations()` where appropriate (e.g., conversationScripts has many scriptLines, scriptLines belongs to conversationScripts).
