- Clerk auth guard
- Accept query param `id` (UUID)
- Delete from `savedVocabulary` where `id` matches AND `userId` matches (prevents deleting other users' vocab)
- Use `db.delete(savedVocabulary).where(and(eq(savedVocabulary.id, id), eq(savedVocabulary.userId, userId)))`
- Return `{ success: true }` with 200
- Wrap in try/catch, return 500 on error

- Return `{ items: [{ id, traditional }] }`

Create `src/hooks/useCharacterPopup.ts` — the central state management hook for the popup.

**State managed:**
- `activeWord: string | null` — currently hovered/tapped word
- `isVisible: boolean` — whether popup is showing
- `lookupData: LookupData | null` — dictionary entries from /api/dictionary/lookup
- `characterData: CharacterDetailData | null` — character detail from /api/dictionary/character (only for single chars)
- `isLoading: boolean` — true while fetching dictionary data
- `error: string | null` — error message from failed fetch
- `savedVocabSet: Map<string, string>` — Map of traditional -> savedId for instant bookmark checks
- `virtualEl: VirtualElement | null` — Floating UI virtual element for positioning

**Type definitions (inline in hook file):**
```typescript
interface DictionaryEntry {
  id: string;
  traditional: string;
  simplified: string;
  pinyin: string;
  pinyinDisplay: string;
  jyutping: string | null;
  definitions: string[];
  source: "cedict" | "canto" | "both";
  isSingleChar: boolean;
}

interface LookupData {
  entries: DictionaryEntry[];
}

interface CharacterDetail {
  character: string;
  pinyin: string[];
  jyutping: string[];
  radical: string;
  radicalMeaning: string | null;
  strokeCount: number;
  decomposition: string;
  etymologyType: string | null;
  etymologyHint: string | null;
  etymologyPhonetic: string | null;
  etymologySemantic: string | null;
  definition: string | null;
  frequencyRank: number | null;
  strokePaths: unknown;
  strokeMedians: unknown;
}

interface CharacterDetailData {
  character: CharacterDetail | null;
  examples: Array<{
    traditional: string;
    simplified: string;
    pinyin: string;
    pinyinDisplay: string;
    definitions: string[];
    source: string;
  }>;
}

type VirtualElement = { getBoundingClientRect: () => DOMRect };
```

**Core logic:**

1. **showPopup(word: string, element: HTMLElement):**
   - Set `activeWord = word`
   - Set `virtualEl = { getBoundingClientRect: () => element.getBoundingClientRect() }`
   - Set `isVisible = true`
   - Trigger dictionary fetch (debounced)

2. **hidePopup():**
   - Use a 200ms setTimeout before actually hiding (to allow cursor to move to popup)
   - Store timeout ID in a ref, cancel on new showPopup call
   - Set `isVisible = false`, `activeWord = null`, `lookupData = null`, `characterData = null`

3. **cancelHide():**
   - Clear the hide timeout (called when cursor enters the popup)

4. **Dictionary fetching** (in a useEffect triggered by activeWord):
   - Use AbortController ref — abort previous request on new word
   - Set `isLoading = true`
   - Fetch `/api/dictionary/lookup?word=${encodeURIComponent(activeWord)}`
   - If word is single character (`[...word].length === 1`), also fetch `/api/dictionary/character?char=${encodeURIComponent(word)}`
   - Use `Promise.all` for parallel fetching when applicable
   - On success: set `lookupData` and `characterData` (only if activeWord still matches — stale check)
   - On error (not abort): set `error`
   - Set `isLoading = false`
   - Use `useDebouncedCallback` from `use-debounce` (already installed) with 150ms delay

5. **Saved vocabulary tracking:**
   - On mount, fetch GET /api/vocabulary to get `{ items: [{ id, traditional }] }`
   - Build a `Map<string, string>` (traditional -> id) stored in state
   - `isSaved(word: string): boolean` — check if word's traditional form is in the map
   - `toggleSave(entry: DictionaryEntry): Promise<void>` — optimistic toggle:
     - If currently saved: DELETE /api/vocabulary?id=X, remove from map
     - If not saved: POST /api/vocabulary with entry data, add to map
     - On error: rollback the map change

**Return type:**
```typescript
{
  activeWord: string | null;
  isVisible: boolean;
  lookupData: LookupData | null;
  characterData: CharacterDetailData | null;
  isLoading: boolean;
  error: string | null;
  virtualEl: VirtualElement | null;
  showPopup: (word: string, element: HTMLElement) => void;
  hidePopup: () => void;
  cancelHide: () => void;
  isSaved: (traditional: string) => boolean;
  toggleSave: (entry: DictionaryEntry) => Promise<void>;
}
```

Use "use client" directive. Follow the same ref guard pattern as useTTS (mountedRef for state updates after async ops). Add `useDebouncedCallback` import from `use-debounce`.
