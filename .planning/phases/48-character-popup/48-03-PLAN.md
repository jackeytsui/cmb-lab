   - Outer wrapper: `div` with Floating UI styles + ref + onMouseEnter={onCancelHide} + onMouseLeave={onHide}

   ┌─ PopupHeader (always, if lookupData exists) ─────────┐

<task type="auto">
  <name>Task 2: Wire CharacterPopup into ReaderClient with hover/click handlers</name>
  <files>src/app/(dashboard)/dashboard/reader/ReaderClient.tsx</files>
  <action>
Update `src/app/(dashboard)/dashboard/reader/ReaderClient.tsx` to integrate the character popup:

1. **Import new dependencies:**
   - `import { useCharacterPopup } from "@/hooks/useCharacterPopup"`
   - `import { CharacterPopup } from "@/components/reader/CharacterPopup"`

2. **Initialize the hook:**
   ```typescript
   const popup = useCharacterPopup();
   ```

3. **Create hover/click handlers:**
   ```typescript
   const handleWordHover = useCallback(
     (word: string, _index: number, element: HTMLElement) => {
       popup.showPopup(word, element);
     },
     [popup.showPopup]
   );

   const handleWordClick = useCallback(
     (word: string, _index: number, element: HTMLElement) => {
       // On click (especially for mobile tap): force show popup
       popup.showPopup(word, element);
     },
     [popup.showPopup]
   );
   ```

4. **Wire handlers to ReaderTextArea:**
   Update the `<ReaderTextArea>` component to pass the new props:
   ```tsx
   <ReaderTextArea
     segments={segments}
     annotationMode={annotationMode}
     fontSize={fontSize}
     onWordHover={handleWordHover}
     onWordClick={handleWordClick}
   />
   ```

5. **Render CharacterPopup:**
   Add `<CharacterPopup>` AFTER `<ReaderTextArea>` in the JSX tree:
   ```tsx
   <CharacterPopup
     isVisible={popup.isVisible}
     virtualEl={popup.virtualEl}
     activeWord={popup.activeWord}
     lookupData={popup.lookupData}
     characterData={popup.characterData}
     isLoading={popup.isLoading}
     error={popup.error}
     isSaved={popup.activeWord ? popup.isSaved(
       popup.lookupData?.entries[0]?.traditional ?? popup.activeWord
     ) : false}
     onToggleSave={() => {
       const entry = popup.lookupData?.entries[0];
       if (entry) popup.toggleSave(entry);
     }}
     onHide={popup.hidePopup}
     onCancelHide={popup.cancelHide}
   />
   ```

6. **Add useCallback import** if not already present (it is already imported in the current file).

7. **Keep existing functionality intact:** The paste, import, toolbar, conversion, segmentation, and annotation mode features must all continue working unchanged. Only add the popup integration — do not modify the existing state flow or component structure.

8. **Build verification:** Run `npm run build` to ensure the full Next.js application builds successfully with the new popup integration. Fix any type errors or import issues.
