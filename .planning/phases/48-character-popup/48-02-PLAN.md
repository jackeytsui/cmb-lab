**StrokeAnimation.tsx** — HanziWriter wrapper with play/pause controls:
- "use client" directive
- Props: `{ character: string }` (single character only)
- Use a container div with `useRef<HTMLDivElement>` for HanziWriter to render into
- Use `useRef<HanziWriter | null>` for the writer instance (import HanziWriter from "hanzi-writer")
- useEffect keyed on `character`:
  - Clear container: `containerRef.current.innerHTML = ""`
  - Create writer: `HanziWriter.create(containerRef.current, character, {...options})`
  - Options: `{ width: 120, height: 120, padding: 5, showOutline: true, strokeColor: "#e4e4e7", radicalColor: "#22d3ee", outlineColor: "#3f3f46", strokeAnimationSpeed: 1, delayBetweenStrokes: 300, renderer: "svg" }`
  - Store in writerRef
  - Cleanup: set writerRef.current = null
- Three control buttons using lucide-react icons:
  - Play (Play icon): calls `writerRef.current?.animateCharacter()`
  - Pause (Pause icon): calls `writerRef.current?.pauseAnimation()`
  - Replay (RotateCcw icon): calls `writerRef.current?.animateCharacter()` (replays from start)
- Track animation state with `isAnimating` boolean (set true on play, false on complete callback)
- Use the `onComplete` callback in animateCharacter options to set isAnimating = false
- Layout: centered container with controls below
- Only render for single characters (guard: `[...character].length === 1`, else show nothing)
