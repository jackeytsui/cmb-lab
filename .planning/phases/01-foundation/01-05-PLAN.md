<task type="auto">
  <name>Task 1: Create API route for user's accessible courses</name>
  <files>src/app/api/courses/route.ts</files>
  <action>
    Create `src/app/api/courses/route.ts` with GET handler that:

    1. Imports: `auth` from `@clerk/nextjs/server`, `db` from `@/db`, schema tables
    2. Gets current user's clerkId via `auth()`
    3. Returns 401 if not authenticated
    4. Queries database to get courses user has access to:
       - Join: users -> courseAccess -> courses
       - Filter: `users.clerkId = userId` AND `courses.deletedAt IS NULL`
       - Filter: `courseAccess.expiresAt IS NULL OR courseAccess.expiresAt > NOW()`
       - Select: course fields + accessTier from courseAccess
    5. Use Drizzle ORM query with proper relations or raw SQL join
    6. Return JSON: `{ courses: Course[] }` with each course having `accessTier` property

    Example query logic (Drizzle):
    ```typescript
    const userCourses = await db
      .select({
        id: courses.id,
        title: courses.title,
        description: courses.description,
        thumbnailUrl: courses.thumbnailUrl,
        accessTier: courseAccess.accessTier,
      })
      .from(courseAccess)
      .innerJoin(users, eq(courseAccess.userId, users.id))
      .innerJoin(courses, eq(courseAccess.courseId, courses.id))
      .where(
        and(
          eq(users.clerkId, clerkId),
          isNull(courses.deletedAt),
          or(
            isNull(courseAccess.expiresAt),
            gt(courseAccess.expiresAt, new Date())
          )
        )
      );
    ```
  </action>
  <verify>
    With valid auth token and seed data: `curl -H "Authorization: Bearer $TOKEN" /api/courses`
    Should return `{ courses: [...] }` array (empty if no access, populated if granted)
  </verify>
  <done>
    API endpoint exists at `/api/courses` returning only courses user has valid access to.
    Expired access grants are excluded.
  </done>
