    - State: deferredPrompt (BeforeInstallPromptEvent | null), isInstalled (boolean), isIOS (boolean)
    - On mount: check window.matchMedia("(display-mode: standalone)") for already-installed

    - On mount: addEventListener("beforeinstallprompt") to capture and prevent default, store in state

    - Import usePWAInstall hook

    - Listen for custom event "pwa-first-lesson-complete" on window to set showPrompt true
    - Also check localStorage key "pwa-install-dismissed" on mount — if set, don't show
    - If isInstalled or dismissed or !showPrompt: return null
    - If isIOS and !isInstalled:
      - Render a toast/banner at bottom of screen (fixed position, z-50)
      - Dark themed card matching app aesthetic (bg-gray-900 border border-gray-800 rounded-lg shadow-xl)

      - Dismiss button (X) that sets localStorage "pwa-install-dismissed" and hides
    - If canPrompt (Chromium):

      - "Install" button (primary style: bg-indigo-600 hover:bg-indigo-500) that calls triggerInstall
