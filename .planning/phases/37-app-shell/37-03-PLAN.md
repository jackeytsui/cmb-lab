       - Label: "Daily XP Goal"
       - Description: "Set your daily learning target"
       - Four tier buttons (grid layout):

         - Serious: 150 XP
         - Intense: 250 XP
       - Each button shows the tier name and XP amount
       - Selected tier has cyan border + bg highlight

       **Section 4: Notification Preferences**
       - Label: "Notification Preferences"
       - Description: "Manage which notifications you receive"
       - Fetch notification categories from `/api/notifications/preferences` on component mount
       - Display each category with a toggle switch (use a styled checkbox or button)
       - Save notification preferences via PATCH to `/api/notifications/preferences` (separate from main save)
       - Handle loading state and error for notification fetch independently
