**TranscriptLine.tsx** -- Replace plain text rendering with segmented WordSpan components:

1. Add imports: `WordSpan` and `AnnotationMode` from `@/components/reader/WordSpan`, `segmentText` from `@/lib/segmenter`, and `useMemo` from react.
2. Add new prop `annotationMode: AnnotationMode` to `TranscriptLineProps` (default: `"plain"`).
3. Inside the component, memoize segmentation: `const segments = useMemo(() => segmentText(text), [text])`.
4. Replace the bare `{text}` in the JSX with a `<span>` wrapper containing mapped `segments.map((seg, i) => <WordSpan key={i} text={seg.text} index={i} isWordLike={seg.isWordLike} annotationMode={annotationMode} />)`.
5. Keep the timestamp `<span>` before the word spans.
6. Add annotation-aware line height: when annotationMode is not "plain", add `leading-[3]` class to the line container (ruby annotations need extra vertical space). Use cn() to conditionally apply.

**TranscriptPanel.tsx** -- Add event delegation for word hover/click and popup-aware auto-scroll:

1. Add new props to `TranscriptPanelProps`:
   - `annotationMode?: AnnotationMode` (import from WordSpan, default `"plain"`)
   - `onWordHover?: (word: string, index: number, element: HTMLElement) => void`
   - `onWordClick?: (word: string, index: number, element: HTMLElement) => void`
   - `isPopupVisible?: boolean` (default `false`)
2. Add the `findWordElement` helper function (same pattern as ReaderTextArea.tsx):
   ```typescript
   function findWordElement(target: EventTarget): HTMLElement | null {
     const el = target as HTMLElement;
     if (!el.closest) return null;
     return el.closest("[data-word]") as HTMLElement | null;
   }
   ```
3. Add `handleMouseOver` and `handleClick` callbacks using `useCallback` that call `findWordElement`, extract `data-word` and `data-index` attributes, and invoke `onWordHover`/`onWordClick` respectively. These handlers go on the scroll container div (event delegation).
4. Modify the auto-scroll useEffect: add `isPopupVisible` to the guard condition. When `isPopupVisible` is true, skip auto-scroll entirely (same pattern as user scroll detection but takes priority).
5. Pass `annotationMode` through to each `TranscriptLine`.
6. Attach `onMouseOver={handleMouseOver}` and `onClick={handleClick}` to the scroll container div. The existing `onScroll` handler stays.
  </action>
