  <name>Task 2: Create PromptForm component for editing</name>
  <files>src/components/admin/PromptForm.tsx</files>
  <action>
Create `src/components/admin/PromptForm.tsx` as a client component:

1. **Props:**
   ```typescript
   interface PromptFormProps {
     prompt: {
       id: string;
       slug: string;
       name: string;
       currentContent: string;
       currentVersion: number;
     };
     onVersionCreated?: (newVersion: number) => void;
   }
   ```

2. **State:**
   - content: string (initialized from prompt.currentContent)
   - changeNote: string (empty)
   - saving: boolean
   - error: string | null
   - success: boolean

3. **Form layout:**
   - Card with header "Edit Prompt"
   - Textarea for content:
     - Full width, min-height 300px
     - font-mono for code-like appearance
     - bg-zinc-800 border-zinc-700
   - Input for change note (optional):
     - Placeholder: "What changed? (optional)"
   - Save button:
     - Disabled while saving or if content unchanged
     - Shows "Saving..." while saving

4. **Save handler:**
   - Set saving = true
   - PUT to /api/admin/prompts/${prompt.id}
   - Body: { content, changeNote }
   - On success:
     - Set success = true, clear after 3s
     - Call onVersionCreated with new version
   - On error: Set error message
   - Set saving = false

5. **Feedback:**
   - Success: Green toast/message "Saved as version X"
   - Error: Red error message

6. **Character count:** Show content.length under textarea

Use existing form patterns from admin components.
  </action>
