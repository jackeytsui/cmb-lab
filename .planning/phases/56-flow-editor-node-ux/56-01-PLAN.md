  <action>
Create a new custom React Flow edge component `DeletableEdge` that wraps the built-in `SmoothStepEdge` (or `BezierEdge`) behavior but adds a delete button.

Implementation details:
- Import `EdgeProps`, `BaseEdge`, `getSmoothStepPath`, `EdgeLabelRenderer`, `useReactFlow` from `@xyflow/react`
- The component receives standard `EdgeProps` plus any custom data
- Use `getSmoothStepPath` to compute the edge path (smooth step matches the existing style)
- Render the edge path via `<BaseEdge>` with the existing animated arrow styling (stroke #6366f1, strokeWidth 2, markerEnd ArrowClosed)
- Use `EdgeLabelRenderer` to position a delete button at the edge midpoint (`labelX`, `labelY` from getSmoothStepPath)
- The delete button:
  - Is a small circle (20x20px) with an X icon (use lucide `X` icon, 12x12)
  - Has `opacity-0 group-hover:opacity-100` classes (invisible until edge hovered)
  - Uses `pointer-events: all` to be clickable
  - Background: white with red border, on hover: red bg with white icon
  - On click: calls `useReactFlow().deleteElements({ edges: [{ id }] })` where `id` is the edge's id
- Wrap the entire edge in a group element with `className="group"` -- BUT since SVG edges and the label renderer are separate DOM trees, use a different approach: track hover state via `selected` prop (React Flow sets this) OR use CSS on the edge wrapper. Recommended approach: use the `selected` prop from EdgeProps -- when `selected` is true OR on hover, show the button. For hover, add `interactionWidth: 30` to edge defaults in FlowEditor and use the `[data-selected="true"]` CSS approach, OR simply always show the button with a subtle opacity and make it fully visible on hover via the EdgeLabelRenderer div.

  Simplest reliable approach: Render the delete button always but with low opacity (opacity-0), and use a `data-testid` div that becomes `opacity-100` when the edge `selected` prop is true. Also add a `:hover` CSS rule on the label container. The EdgeLabelRenderer content is normal HTML so standard hover works.

  Structure:
  ```tsx
  export default function DeletableEdge({ id, sourceX, sourceY, targetX, targetY, sourcePosition, targetPosition, style, markerEnd, selected, ...props }: EdgeProps) {
    const { deleteElements } = useReactFlow();
    const [edgePath, labelX, labelY] = getSmoothStepPath({ sourceX, sourceY, targetX, targetY, sourcePosition, targetPosition });

    return (
      <>
        <BaseEdge path={edgePath} markerEnd={markerEnd} style={style} />
        <EdgeLabelRenderer>
          <div style={{ position: 'absolute', transform: `translate(-50%, -50%) translate(${labelX}px, ${labelY}px)`, pointerEvents: 'all' }}
               className={`transition-opacity ${selected ? 'opacity-100' : 'opacity-0 hover:opacity-100'}`}>
            <button onClick={(e) => { e.stopPropagation(); deleteElements({ edges: [{ id }] }); }}
                    className="w-5 h-5 rounded-full bg-white border border-red-300 flex items-center justify-center hover:bg-red-500 hover:border-red-500 text-red-400 hover:text-white shadow-sm transition-all cursor-pointer"
                    title="Delete connection">
              <X className="w-3 h-3" />
            </button>
          </div>
        </EdgeLabelRenderer>
      </>
    );
  }
  ```

- Export as default. Mark as "use client" at top.
- Import `X` from lucide-react.
