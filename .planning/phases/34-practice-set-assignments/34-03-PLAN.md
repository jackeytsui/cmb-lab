  <action>
Create a "use client" component that receives resolved assignments and renders them with filter controls.

**Props:**
```typescript
import type { ResolvedAssignment } from "@/lib/assignments";

interface PracticeDashboardProps {
  assignments: ResolvedAssignment[];
}
```

**State:**
- `statusFilter`: 'all' | 'pending' | 'completed' (default 'all')
- `sortBy`: 'due_date' | 'assigned_date' | 'title' (default 'due_date')

**Filtering logic (useMemo):**
1. Filter by status if not 'all'
2. Sort:
   - 'due_date': assignments with due dates first (ascending), then those without due dates
   - 'assigned_date': by assignedAt descending (newest first)
   - 'title': alphabetical by practiceSetTitle
3. Return filtered + sorted array

**UI Layout:**

1. **Filter bar** (flex row, gap-3, mb-6):
   - Status filter: Three buttons (All, Pending, Completed) styled as segmented control. Active button has bg-zinc-700, inactive has bg-zinc-800/50. Use emerald dot for completed count, amber dot for pending count.
   - Sort dropdown: `<Select>` with options "Due Date", "Date Assigned", "Title"
   - Summary text: "Showing X of Y assignments"

2. **Assignment cards grid** (grid cols-1 md:cols-2 lg:cols-3 gap-4):
   Each card is a Link to `/practice/${assignment.practiceSetId}`:
   - Card: rounded-lg border border-zinc-700 bg-zinc-800/50 p-4 hover:border-zinc-500 transition-colors
   - Top row: Practice set title (font-semibold text-white) + status badge (green "Completed" with check or zinc "Pending")
   - Middle: Description (text-sm text-zinc-400, line-clamp-2) or "No description" in italic
   - Bottom row (flex justify-between items-center text-xs text-zinc-500):
     - Left: Target type badge (same colors as AssignmentList: blue=course, purple=module, amber=lesson, green=student, rose=tag)
     - Right: Due date formatted with date-fns (`format(dueDate, "MMM d, yyyy")`) or "No due date"
   - If completed: show score (text-emerald-400 "Score: XX%")
   - If due date is past and status is pending: show overdue indicator (text-red-400 "Overdue")

3. **Empty state** (when assignments.length === 0):
   - ClipboardList icon (lucide), large, zinc-500
   - "No practice assignments yet"
   - "Your coach hasn't assigned any practice sets yet. Check back later!"

4. **Filtered empty state** (when filtered.length === 0 but assignments.length > 0):
   - "No assignments match your filters"
   - "Try changing the status filter or sort order"

**Imports:**
- Link from next/link
- format, isPast from date-fns
- Select, SelectContent, SelectItem, SelectTrigger, SelectValue from @/components/ui/select
- ClipboardList, Check, Clock, AlertTriangle from lucide-react
