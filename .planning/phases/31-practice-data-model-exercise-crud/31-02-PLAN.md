  <name>Task 1: Create CRUD library helpers and parseBlankSentence utility</name>
  <files>src/lib/practice.ts</files>
  <action>
Create `src/lib/practice.ts` with database query helpers following the pattern from `src/lib/interactions.ts` and `src/lib/tags.ts`:

**Imports:**
- `db` from "@/db"
- `practiceSets, practiceExercises` from "@/db/schema"
- `eq, isNull, asc, and, desc` from "drizzle-orm"
- `type ExerciseDefinition` from "@/types/exercises"

**Practice Set helpers:**
- `createPracticeSet(data: { title: string; description?: string; createdBy: string })`: Insert into practiceSets with status "draft", return the created set.
- `updatePracticeSet(id: string, data: { title?: string; description?: string; status?: "draft" | "published" | "archived" })`: Update set by ID where deletedAt IS NULL, return updated set.
- `deletePracticeSet(id: string)`: Soft delete -- set deletedAt = new Date() where id matches and deletedAt IS NULL. Return updated row.
- `listPracticeSets(options?: { status?: string })`: Select all sets where deletedAt IS NULL, optionally filter by status, order by createdAt DESC. Return array.
- `getPracticeSet(id: string)`: Select set by ID where deletedAt IS NULL. Return single set or null.

**Exercise helpers:**
- `createExercise(data: { practiceSetId: string; type: string; language: string; definition: ExerciseDefinition; sortOrder?: number })`: Insert into practiceExercises, return created exercise.
- `updateExercise(id: string, data: { language?: string; definition?: ExerciseDefinition; sortOrder?: number })`: Update exercise by ID where deletedAt IS NULL. Note: type cannot be changed after creation. Return updated exercise.
- `deleteExercise(id: string)`: Soft delete -- set deletedAt = new Date(). Return updated row.
- `listExercises(practiceSetId: string)`: Select exercises where practiceSetId matches AND deletedAt IS NULL, order by sortOrder ASC. Return array.
- `getExercise(id: string)`: Select exercise by ID where deletedAt IS NULL. Return single exercise or null.

**Utility:**
- `parseBlankSentence(template: string): Array<{ type: "text"; value: string } | { type: "blank"; index: number }>`: Parse a sentence with `{{blank}}` placeholders into an array of text and blank segments. Split by regex `/\{\{blank\}\}/g`. Map alternating segments to text/blank types. Blank index is 0-based sequential. This utility will be shared between the form preview (Plan 05) and the student player (Phase 33).

Export all functions.
  </action>

**`src/app/api/admin/exercises/route.ts`:**

GET handler:
- Check `hasMinimumRole("coach")` -- return 403 if not
- Read `practiceSetId` from searchParams (required, return 400 if missing)
- Call `listExercises(practiceSetId)` from lib/practice
- Return `{ exercises: [...] }`
- Wrap in try/catch, return 500 on error

POST handler:
- Check `hasMinimumRole("coach")` -- return 403 if not
- Parse request body
- Validate `body.definition` with `exerciseDefinitionSchema.safeParse()` -- return 400 with first issue message if invalid
- Validate that `body.definition.type` matches `body.type` (consistency check) -- return 400 if mismatch
- Validate body.practiceSetId exists (return 400 if missing)
- Validate body.language is one of "cantonese" | "mandarin" | "both" (return 400 if invalid)
- Call `createExercise({ practiceSetId, type: result.data.type, language, definition: result.data, sortOrder })`
- Return `{ exercise }` with status 201
- Wrap in try/catch, return 500 on error

**`src/app/api/admin/exercises/[exerciseId]/route.ts`:**

GET handler:
- Check `hasMinimumRole("coach")`
- Extract exerciseId from params
- Call `getExercise(exerciseId)`
- Return 404 if null, else return `{ exercise }`

PUT handler:
- Check `hasMinimumRole("coach")`
- Extract exerciseId from params
- Parse body
- If body.definition provided, validate with `exerciseDefinitionSchema.safeParse()` -- return 400 if invalid
- Call `updateExercise(exerciseId, { language, definition: validatedDef, sortOrder })`

DELETE handler:

All handlers: wrap in try/catch with 500 error response. Use `NextRequest` and `NextResponse` from "next/server".
