  <action>
Create API routes for admin content management. All routes require admin role check using hasMinimumRole("admin").

**courses/route.ts:**
- GET: List all courses (including unpublished), ordered by sortOrder. Exclude soft-deleted (deletedAt IS NULL).
- POST: Create course with {title, description?, thumbnailUrl?, isPublished?, previewLessonCount?, sortOrder?}

**courses/[courseId]/route.ts:**
- GET: Single course with nested modules and lessons (use Drizzle relations)
- PUT: Update course fields (partial update supported)
- DELETE: Soft delete by setting deletedAt = new Date()

**modules/route.ts:**
- GET: List modules for courseId (query param), ordered by sortOrder
- POST: Create module with {courseId, title, description?, sortOrder?}

**modules/[moduleId]/route.ts:**
- GET: Single module with lessons
- PUT: Update module fields
- DELETE: Soft delete

**modules/reorder/route.ts:**
- PATCH: Accept array of {id: string, sortOrder: number} objects
- Update sortOrder for each module in a transaction
- Validate all module IDs belong to same course
- Return 200 with updated modules

**lessons/route.ts:**
- GET: List lessons for moduleId (query param), ordered by sortOrder
- POST: Create lesson with {moduleId, title, description?, muxPlaybackId?, muxAssetId?, durationSeconds?, sortOrder?}

**lessons/[lessonId]/route.ts:**
- GET: Single lesson (for edit form)
- PUT: Update lesson fields
- DELETE: Soft delete

**lessons/reorder/route.ts:**
- PATCH: Accept array of {id: string, sortOrder: number} objects
- Update sortOrder for each lesson in a transaction
- Validate all lesson IDs belong to same module
- Return 200 with updated lessons

Use existing pattern from coach API routes: `await hasMinimumRole("admin")` check at start, return 403 if fails.
Return proper HTTP status codes: 200 for success, 201 for create, 400 for validation, 403 for auth, 404 for not found.
  </action>
  <verify>
Test with curl or Postman:
- POST /api/admin/courses with {title: "Test Course"} returns 201
- GET /api/admin/courses returns array including new course
- PUT /api/admin/courses/{id} with {title: "Updated"} returns 200
- DELETE /api/admin/courses/{id} returns 200
- PATCH /api/admin/modules/reorder with [{id: "...", sortOrder: 1}, {id: "...", sortOrder: 2}] returns 200
- PATCH /api/admin/lessons/reorder with [{id: "...", sortOrder: 1}] returns 200
  </verify>
