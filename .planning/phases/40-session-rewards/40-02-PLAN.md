       - `show()` callback: if hasFired.current, return early. Set hasFired to true, setIsVisible(true). If !shouldReduceMotion and tier !== "keep_practicing", setTimeout to fire confetti at CELEBRATION_TIMING.confettiDelayMs.

       - `reset()` callback: Set hasFired.current = false, setIsVisible(false), confetti.reset(). This method allows callers (e.g., PracticePlayer retry) to reset the hook so it can fire again on the next completion.
       - useEffect cleanup: return () => confetti.reset() (cleans up on unmount/navigation)
       - Return: { isVisible, show, dismiss, reset, tier, shouldReduceMotion }
