SECURITY NOTICE

================================================================================
SETUP INSTRUCTIONS
================================================================================

1. Clone repository:
   git clone <your-repo-url>
   cd New-LMS

2. Install dependencies:
   npm install

3. Create local env file:
   cp .env.example .env.local

4. Populate `.env.local` with your own keys.

5. Run migrations:
   npm run db:migrate

6. Start dev server:
   npm run dev
